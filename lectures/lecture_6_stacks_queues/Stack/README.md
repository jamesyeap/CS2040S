Data in a Stack is accessed in a last-in-first-out (LIFO) manner.

# Operations (3 main ones)
- push
- pop
- peek

# Uses of a Stack
- Calling a function
- Recursion
- Matching parentheses
	- are the brackets properly closed?
	- example
	- `“[()]{}{[()()]()}”` is OKAY
	- `“[(])”` is NOT OKAY
- Evaluating arithmetic expressions
- Traversing a maze
